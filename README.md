# Online App Hub

A unified web application built using Flask, HTML, CSS, and JavaScript.

## Features
- 🛒 E-commerce module
- 💬 Messaging system
- 🎵 Music hub
- 🏋️ Fitness tracker
- 📚 Study hub
- 🌐 Social media features

## Technologies Used
- Python Flask
- HTML5
- CSS3
- JavaScript
- SQLite/MySQL (if used)

## How to Run

1. Clone the repository:
git clone https://github.com/sharonlilly05/online-app-hub.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the application:
python app.py

4. Open:
http://127.0.0.1:5000
